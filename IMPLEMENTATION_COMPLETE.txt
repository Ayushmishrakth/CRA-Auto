================================================================================
                            API ENDPOINT CHANGES
================================================================================

NEW/UPDATED ENDPOINT:
  POST /reports/assessments/{assessment_id}/generate

PARAMETERS:
  - logo (file, optional)          PNG/JPG/SVG, max 5MB
  - company_name (string, optional) Max 200 characters
  - company_address (string, optional) Max 500 characters
  - report_format (string, required) "pdf" | "docx" | "both"

RESPONSE:
  - Format "pdf"  → PDF file download
  - Format "docx" → Word document download
  - Format "both" → ZIP archive with both formats

ERROR HANDLING:
  - 400 Bad Request (invalid logo, size exceeded, wrong format)
  - 401 Unauthorized (missing/invalid token)
  - 404 Not Found (assessment doesn't exist)
  - 500 Internal Server Error (generation failed, with fallback)
